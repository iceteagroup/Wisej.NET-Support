# Wisej.NET Support

[![Open Wisej Support](https://img.shields.io/badge/Open%20Wisej%20Support-wisej.com%2Fsupport-2ea44f?style=for-the-badge)](https://wisej.com/support/)

Welcome to the public Wisej.NET support repository.

Use this repository to ask questions, report issues, request improvements, and find answers from previous Wisej.NET support discussions. The issue tracker is connected to the [Wisej.NET support page](https://wisej.com/support/) and is intended to be a searchable knowledge base for the community.

## Before Opening an Issue

Please search existing issues first. Many questions from the previous Wisej support forum have been migrated here, including their original discussion history, so an answer may already be available.

Also check the [Wisej.NET documentation](https://docs.wisej.com/) for product guides, API details, and setup information.

## What to Include

Clear, reproducible reports are easier to answer and more useful to future readers. When opening a support issue, include:

* The Wisej.NET version you are using
* A concise description of the problem or request
* What you expected to happen
* What actually happened
* A small runnable test case when reporting a bug
* Relevant browser, operating system, Visual Studio, .NET, or deployment details

Do not post passwords, license keys, access tokens, private customer data, or other confidential information.

## Issue Types

Use the support issue template and choose the category that best fits your post:

* General
* Bug
* New Feature
* Suggestion

## Community Guidelines

This is a public support space. Please keep posts focused on Wisej.NET, write in English, use descriptive titles, and follow the [Code of Conduct](CODE_OF_CONDUCT.md).
