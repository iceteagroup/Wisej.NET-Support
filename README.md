# Wisej.NET Support

[![Open Wisej Support](https://img.shields.io/badge/Open%20Wisej%20Support-wisej.com%2Fsupport-2ea44f?style=for-the-badge)](https://wisej.com/support/)

This repository is used by the [wisej.com/support](https://wisej.com/support/) page to replace the old Wisej support forum.

Support discussions from the previous Sabai WordPress forum are imported here as GitHub issues. Imported issues preserve the original question and answer timestamps in the issue body and comments.

The local import ledger is stored in `imported-sabai-issues.json`; reruns use that file to skip Sabai issues that were already imported.
